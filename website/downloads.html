# Create a new contract solution

Recommended Installation

Install as .NET Global Tool (Recommended)

The easiest way to install RNCC is as a .NET global tool:

# Install RNCC globally
dotnet tool install -g rncc --version 1.0.2

# Verify installation
rncc --version

# Create your first solution
rncc new MyContract --template=solution

This method ensures RNCC is always available in your PATH and easily updatable.

Alternative Installation Methods

# Linux/macOS - Download and install in one command
curl -L https://github.com/r3e-network/r3e-devpack-dotnet/releases/latest/download/rncc-linux-x64 -o rncc
chmod +x rncc
sudo mv rncc /usr/local/bin/
# Linux - Using wget
wget https://github.com/r3e-network/r3e-devpack-dotnet/releases/latest/download/rncc-linux-x64
chmod +x rncc-linux-x64
sudo mv rncc-linux-x64 /usr/local/bin/rncc
# Windows PowerShell - Download to current directory
Invoke-WebRequest -Uri "https://github.com/r3e-network/r3e-devpack-dotnet/releases/latest/download/rncc-win-x64.exe" -OutFile "rncc.exe"

# Add to PATH (requires admin privileges)
$path = [Environment]::GetEnvironmentVariable("PATH", "User")
[Environment]::SetEnvironmentVariable("PATH", "$path;$PWD", "User")
# macOS - Coming soon via Homebrew
# brew tap r3e-network/r3e
# brew install rncc
