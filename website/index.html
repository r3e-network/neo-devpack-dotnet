Quick Start

1

Download RNCC

Get the compiler for your platform

# Linux/macOS
wget https://github.com/r3e-network/r3e-devpack-dotnet/releases/latest/download/rncc-linux-x64
chmod +x rncc-linux-x64
2

Write Your Contract

Create a simple smart contract

using Neo.SmartContract.Framework;

public class HelloWorld : SmartContract
{
    public static string SayHello(string name)
    {
        return $"Hello, {name}!";
    }
}
3

Compile & Deploy

Compile with optimization and WebGUI

# Compile with all features
./rncc-linux-x64 HelloWorld.cs \
  --optimize=All \
  --generate-webgui \
  --security-analysis
